# EDIT SECTION

## Introduction

This section allows users to define changes to be applied after OPM Flow has processed the data in the [GRID](#__RefHeading___Toc38674_784232322) section, that is the resulting pore volume ([PORV](#__RefHeading___Toc96547_718313858)) and transmissibility arrays ([TRANX](#__RefHeading___Toc93085_718313858), [TRANY](#__RefHeading___Toc93087_718313858), and [TRANZ](#__RefHeading___Toc93089_718313858)). The entered primary static arrays ([PORO](#__RefHeading___Toc45797_719036256), [PERMX](#__RefHeading___Toc45791_719036256), etc.) in the [GRID](#__RefHeading___Toc38674_784232322) section are no longer available and all modifications are applied to the pore volume and transmissibility arrays.

Historically the intention of this section was allow for the editing of the processed data; however, the features available in this section have, through time, migrated to the [GRID](#__RefHeading___Toc38674_784232322) section. For example the array operator keywords like [ADD](#__RefHeading___Toc4412_421927891), [COPY](#__RefHeading___Toc45761_719036256), [MULTIPLY](#__RefHeading___Toc296609_1576177388), etc. are available in the [GRID](#__RefHeading___Toc38674_784232322) section and thus enable editing of the primary static arrays ([PORO](#__RefHeading___Toc45797_719036256), [PERMX](#__RefHeading___Toc45791_719036256), etc.). This increased capability in the [GRID](#__RefHeading___Toc38674_784232322) section has therefore made the [EDIT](#__RefHeading___Toc40641_784232322) section somewhat redundant.

Nevertheless the section is sometimes used by users to incorporate history matching parameter changes and by static earth modeling software packages to import the pore volumes and transmissibilities calculated in the static model directly into the numerical model via the [EDIT](#__RefHeading___Toc40641_784232322) section. Although the latter work flow is not very common.
