As the primary purpose of this section is to modify the simulator's calculated pore volumes and transmissibilities, then the properties used to define these arrays must have been fully defined in the GRID section. The arrays available for modification in the EDIT section are listed in together with the associated GRID arrays used to generate the EDIT property array.
