## <a id="-refheading-toc357954-387840558"></a>Data Requirements

There are no data requirements for the GLOBAL section keywords, as all the keywords are optional and considered to be utility keywords that can be used throughout the input deck. A complete list of these global keywords is given in Table 4.1.
| No. | GLOBAL Keyword | Description | Section Applied |
| --- | --- | --- | --- |
| 1 | [COLUMNS](#-refheading-toc258603-3519154785) | [COLUMNS – Define Input File Column Margins](#411columns-define-input-file-column-margins). The [COLUMNS](#-refheading-toc258603-3519154785) keyword defines the input file column margins; characters outside the margins are ignored by the input parser. | ALL |
| 2 | [DEBUG](#-refheading-toc52485-2479612490) | [DEBUG – Define the Debug Data to be Printed to File](#311debug-define-the-debug-data-to-be-printed-to-file). This keyword defines the debug data to be written to the debug file \(\*.DBG\), it is ignored by OPM Flow. | ALL |
| 3 | [ECHO](#-refheading-toc52483-2479612490) | [ECHO – Activate Echoing of User Input Files to the Print File](#312echo-activate-echoing-of-user-input-files-to-the-print-file). Turns on echoing of all the input files to the print file. | ALL |
| 4 | [END](#-refheading-toc46631-2479612490) | [END – Define the End of the Input File](#312end-defines-the-end-of-the-input-file). This keyword marks the end of the input file and can occur in any section. Any keywords and data after the [END](#-refheading-toc46631-2479612490) keyword are ignored. | ALL |
| 5 | [ENDINC](#-refheading-toc51671-2479612490) | [ENDINC – Define the End of an Include File](#314endinc-defines-the-end-of-an-include-file). This keyword marks the end of an include file specified on the [INCLUDE](#-refheading-toc55749-2479612490) keyword. When the [ENDINC](#-refheading-toc51671-2479612490) keyword is encountered in the INCLUDE file, input data is read from the next keyword in the current file. | ALL |
| 6 | [ENDSKIP](#-refheading-toc52491-2479612490) | [ENDSKIP – Deactivate Skipping of Keywords and Input Data](#315endskip-deactivates-skipping-of-keywords-and-input-data). Turns off skipping of keywords that was activated by the [SKIP](#-refheading-toc52489-2479612490), [SKIP100](#-refheading-toc65046-1640804870), or [SKIP300](#-refheading-toc980121-1781444514) keywords. | ALL |
| 7 | [EXTRAPMS](#-refheading-toc45777-719036256) | [EXTRAPMS – Activate Extrapolation Warning Messages](#316extrapms-activate-extrapolation-warning-messages). The [EXTRAPMS](#-refheading-toc45777-719036256) keyword activates extrapolation warning messages for when simulator extrapolates the PVT or VFP tables. | ALL |
| 8 | [FORMFEED](#-refheading-toc322374-803326780) | [FORMFEED – Defined the Print File Form\-Feed Character](#418formfeed-defined-the-print-file-form-feed-character). The keyword defines the form\-feed character, or carriage control character, for the output print  \(\*.PRT\) run summary \(\*.RSM\) files. | ALL |
| 9 | [INCLUDE](#-refheading-toc55749-2479612490) | [INCLUDE – Load Another Data File at the Current Position](#317include-load-another-data-file-at-the-current-position). The [INCLUDE](#-refheading-toc55749-2479612490) keyword informs OPM Flow to continue reading input data from the specified INCLUDE file. | ALL |
| 10 | [MESSAGE](#-refheading-toc551671-3181922006) | [MESSAGE – Output User Message](#4110messages-output-user-message). The [MESSAGE](#-refheading-toc551671-3181922006) keyword outputs a user message to the terminal, as well as to the print \(\*.PRT\) and debug  \(\*.DBG\) files. | ALL |
| 11 | [MESSAGES](#-refheading-toc61634-2479612490) | [MESSAGES – Define Message Print Limits and Stop Limits](#317messages-defines-message-print-limits-and-stop-limits). The [MESSAGES](#-refheading-toc61634-2479612490) keyword defines the print and stops levels for various messages. | ALL |
| 12 | [NOECHO](#-refheading-toc52487-2479612490) | [NOECHO – Deactivate Echoing of User Input Files to the Print File](#317noecho-deactivate-echoing-of-user-input-files-to-the-print-file). Turns off echoing of all the input files to the print file. | ALL |
| 13 | [NOWARN](#-refheading-toc54928-2479612490) | [NOWARN – Deactivate Warning Messages](#318nowarn-deactivate-warning-messages). Turns off warning messages to be printed to the print file. | ALL |
| 14 | [SKIP](#-refheading-toc52489-2479612490) | [SKIP – Activate Skipping of All Keywords and Input Data](#3111skip-activate-skipping-of-all-keywords-and-input-date). The keyword turns on skipping of keywords until the [ENDSKIP](#-refheading-toc52491-2479612490) activation keyword is encountered. | ALL |
| 15 | [SKIP100](#-refheading-toc65046-1640804870) | [SKIP100 – Activate Skipping of Keywords by Black\-Oil Simulator](#3232skip100-activate-skipping-of-black-oil-keywords-and-input-date). This keyword turns on skipping of all keywords and input data by the commercial black\-oil simulator until the [ENDSKIP](#-refheading-toc52491-2479612490) keyword is encountered. | ALL |
| 16 | [SKIP300](#-refheading-toc980121-1781444514) | [SKIP300 – Activate Skipping of Compositional Keywords](#3113skip300-activate-skipping-of-compositional-keywords-and-input-date). This keyword turns on skipping of all keywords and input data by the commercial compositional simulator until the [ENDSKIP](#-refheading-toc52491-2479612490) keyword is encountered. | ALL |
| 17 | [WARN](#-refheading-toc54930-2479612490) | [WARN – Activate Warning Messages](#3112warn-activate-warning-messages). Turns on warning messages to be printed to the print file \(\*.PRT\); note that this keyword is activated by default. | ALL |
| Notes: - Note that not all keywords and features listed above are implemented in OPM Flow.  Cells not colored in the No. column indicate the keyword is supported or partially supported by OPM Flow, cells colored gray indicate that the keyword is not applicable, and finally, cells colored in orange indicate keywords that are not supported by OPM Flow and will be ignored by the simulator. |  |  |  |

Table 4.1: Alphabetic List of GLOBAL Section Keywords
