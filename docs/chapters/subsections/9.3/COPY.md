### COPY -- Copy Array Data to Another Array

The [COPY](#__RefHeading___Toc45761_719036256) keyword copies an array (or part of an array) to another array or part of an array. The arrays can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [COPY](#__RefHeading___Toc45761_719036256) keyword is being used.

See [[COPY -- Copy Array Data to Another Array](#anchor-1)](#4.3.11.COPY|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
