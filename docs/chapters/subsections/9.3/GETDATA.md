### GETDATA -- Load and Assign Data Array from INIT or RESTART File

The [GETDATA](#__RefHeading___Toc300022_156692946) keyword loads a data array from a previously generated [INIT](#__RefHeading___Toc45789_719036256) or [RESTART](#__RefHeading___Toc135629_1317547213) file and assigns the loaded array to either same array in the run or another array name.

See [[GETDATA -- Load and Assign Data Array from INIT or RESTART Files](#anchor-1)](#6.3.64.GETDATA – Load and Assign Data Array from RESTART File|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
