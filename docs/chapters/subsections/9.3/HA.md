### HA -- History Match End-Point Gradient Additive Modifier

The [HA](#__RefHeading___Toc279266_870710203) series of keywords defines the history match end-point gradient parameters used to set the additive cumulative end point data, for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

See [[HA -- History Match End-Point Gradient Additive Modifier](#anchor-2)](#8.3.65.HA – History Match End-Point Gradient Parameters (Additive)|outline) in the [PROPS](#__RefHeading___Toc39329_784232322) section for a complete description.
