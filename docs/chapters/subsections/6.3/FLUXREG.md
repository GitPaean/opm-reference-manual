### FLUXREG -- Define Active Flux Regions

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The FLUXREG is used in conjunction with the USEFLUX keyword in runs with have multiple flux regions, to reduce the number of flux regions, that is the keyword specifies which flux regions are active and which are not in the current run.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
