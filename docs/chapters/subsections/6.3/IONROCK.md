### IONROCK -- Define the Ion Exchange Capacity for All the Cells

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The IONROCK keyword defines the ion exchange capacity for all the cells in the model, for when the brine phase has been activated by the BRINE keyword and the Multi-Component Brine model, that allows for the water phase to have multiple water salinities, has been activated by the ECLMC keyword. Both keywords are in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
