#### Description

The [ADDZCORN](#__RefHeading___Toc192779_2366741785) keyword adds a constant to the [ZCORN](#__RefHeading___Toc45757_7190362561) array or part of the array based on cells defined in the specified input box. The constant can be real or integer and can be negative or positive.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
