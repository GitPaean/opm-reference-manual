#### Example

The example sets the mechanical dispersivity for all cells in the model to 1.0.

\--

\-- SET MECHANICAL DISPERSIVITY FOR ALL CELLS (OPM FLOW KEYWORD)

\--

DISPERC

1000\*1.0 /
