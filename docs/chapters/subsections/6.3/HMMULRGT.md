#### Description

[HMMULRGT](#__RefHeading___Toc219822_373485663) defines the transmissibility between two regions gradient parameters, for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

See also the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section that specifies the dimensions for the gradient option, including the maximum number of gradient regions that can be used with the History Match Gradient option.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
