### EQLZCORN -- Modify the Depth of the Corner-Point Depth Array

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The EQLZCORN keyword modifies the depth of a corner point of a grid block on the pillars defining the reservoir grid. The keyword can be only used be used with Irregular Corner-Point Grids.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
