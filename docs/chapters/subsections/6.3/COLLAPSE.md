#### Description

The [COLLAPSE](#__RefHeading___Toc258600_3519154785) keyword defines the which cells can be collapsed in a collapsed Vertical Equilibrium ("VE") run when the [VE](#__RefHeading___Toc557797_487874538) option has been invoked via the [VE](#__RefHeading___Toc557797_487874538) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
