### COLLAPSE -- Define Compressed Vertical Equilibrium Cells

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The COLLAPSE keyword defines the which cells can be collapsed in a collapsed Vertical Equilibrium ("VE") run when the VE option has been invoked via the VE keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
