### DIFFMR- -- Define Grid Block Negative Radial Direction Diffusivity Multipliers

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The DIFFMR- keyword defines the negative radial direction diffusivity multipliers for cells in the current input box for when the Diffusivity option has been activated by the DIFFUSE keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
