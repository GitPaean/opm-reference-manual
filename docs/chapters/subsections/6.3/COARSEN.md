#### Description

The [COARSEN](#__RefHeading___Toc282685_3519154785) keyword defines how a set of cells should be coarsened for when the Local Grid Refinement ("LGR") option has been activated by [LGR](#__RefHeading___Toc55049_4106839650) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
