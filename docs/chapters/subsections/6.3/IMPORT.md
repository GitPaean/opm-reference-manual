#### Description

The [IMPORT](#__RefHeading___Toc539691_2135714711) keyword informs the simulator to import data from the specified [IMPORT](#__RefHeading___Toc539691_2135714711) file. When the end of the [IMPORT](#__RefHeading___Toc539691_2135714711) file is reached, input data is read from the next keyword in the current file. Normally [IMPORT](#__RefHeading___Toc539691_2135714711) files are generated by grid pre-processing software and the keyword allows for both formatted and unformatted (binary) files to be loaded. The keyword can be used to import any valid grid arrays within a section, for example the [EQLNUM](#__RefHeading___Toc73734_2752266063) array in the [REGIONS](#__RefHeading___Toc40648_784232322) section

#### Examples

The first example shown below loads the grid file from the same directory as the data file.

\--

\-- LOAD A IMPORT FILE

\--

IMPORT

\'NOR-OPM-A00-GRID.EGRID\' /

The next example loads the same file one directory above from where the data file is located.

\--

\-- LOAD A IMPORT FILE

\--

IMPORT

\'../NOR-OPM-A00-GRID.EGRID\' /

The final example loads the same file from a separate include subdirectory in the parent directory of the data file.

\--

\-- LOAD A IMPORT FILE

\--

IMPORT

\'../INCLUDE/NOR-OPM-A00-GRID.EGRID\' /
