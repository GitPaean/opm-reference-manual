### GETDATA -- Load and Assign Data Array from INIT or RESTART Files

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The GETDATA keyword loads a data array from a previously generated INIT or RESTART file and assigns the loaded array to either same array in the run or another property array.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
