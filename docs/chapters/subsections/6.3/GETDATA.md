#### Description

The [GETDATA](#__RefHeading___Toc300022_156692946) keyword loads a data array from a previously generated [INIT](#__RefHeading___Toc45789_719036256) or [RESTART](#__RefHeading___Toc135629_1317547213) file and assigns the loaded array to either same array in the run or another property array.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
