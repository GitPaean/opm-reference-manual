#### Description

The [PARAOPTS](#__RefHeading___Toc258883_2928331029) keyword defines various options for parallel runs, for when the Parallel option has been invoked by the [PARALLEL](#__RefHeading___Toc88962_4106839650) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.

See section [Running OPM Flow From The Command Line](#running-opm-flow-from-the-command-line) on how to run OPM Flow in parallel mode.
