### RADFIN4 -- Define a Radial Local Grid Refinement with Four Columns

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword defines a radial local grid refinement using four columns. Local grid refinement is currently not supported by OPM Flow.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
