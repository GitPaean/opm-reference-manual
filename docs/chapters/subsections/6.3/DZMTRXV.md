#### Description

The [DZMATRIX](#__RefHeading___Toc267160_1772380413) keyword defines the matrix block height for the gravity drainage model by grid block for when the Dual Permeability or Dual Porosity models are activated by the [DUALPERM](#__RefHeading___Toc241171_1772380413) and [DUALPORO](#__RefHeading___Toc241173_1772380413) keywords and the Gravity Drainage option is invoked via the [GRAVDR](#__RefHeading___Toc455285_1414963541) and [GRAVDRM](#__RefHeading___Toc471678_1414963541) keywords. All keywords are in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

The keyword is identical to the [DZMATRIX](#__RefHeading___Toc267160_1772380413) keyword in the [GRID](#__RefHeading___Toc38674_784232322) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
