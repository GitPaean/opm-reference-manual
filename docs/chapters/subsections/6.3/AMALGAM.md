#### Description

The [AMALGAM](#__RefHeading___Toc397020_3174375363) keyword defines a Cartesian Local Grid Refinements ("LGR") amalgamations, that is merging several LGRs into one amalgamated [LGR](#__RefHeading___Toc55049_4106839650).

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
