#### Description

The [DOMAINS](#__RefHeading___Toc189519_1772380413) keyword defines the parallel domain properties for when parallel processing has been invoked by the [PARALLEL](#__RefHeading___Toc88962_4106839650) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.

See section [[2.2](#anchor-2)](#2.2.Running OPM Flow 2018-10 |outline)[](#2.2.Running OPM Flow 2018-10 |outline)[[Running OPM Flow 2023-04 From The Command Line](#anchor-2)](#2.2.Running OPM Flow 2018-10 |outline) on how to run OPM Flow in parallel mode.
