#### Description

The [APIGROUP](#__RefHeading___Toc148342_637204261) keyword defines the maximum number of groups of oil PVT tables when the [API](#__RefHeading___Toc4422_421927891) tracking option has been activated via the [API](#__RefHeading___Toc4422_421927891) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
