#### Description

The [MICPPARA](#__RefHeading___Toc394764_111689907) keyword used to define the model parameters for when the [MICP](#__RefHeading___Toc383375_111689907) model had been activated via the [MICP](#__RefHeading___Toc383375_111689907) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

The parameters in this keyword are now defined by the [BIOFPARA](#REF_HEADING_KEYWORD_BIOFPARA) and [PERMFACT](#__RefHeading___Toc712146_1466963378) keywords.
