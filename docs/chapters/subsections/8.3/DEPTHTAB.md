#### Description

This keyword, [DEPTHTAB](#__RefHeading___Toc308919_1539708736), defines the river time and depth tables, for when the River option has been activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
