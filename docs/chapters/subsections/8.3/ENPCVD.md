### ENPCVD -- Define Maximum Capillary Pressure versus Depth Functions

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword defines the maximum gas-oil and water-oil capillary pressure values versus depth for when the end-point scaling option has been invoked by the ENDSCALE keyword in the RUNSPEC section

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
