#### Description

This keyword defines the maximum gas-oil and water-oil capillary pressure values versus depth for when the end-point scaling option has been invoked by the [ENDSCALE](#__RefHeading___Toc68146_2267116897) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
