#### Description

The [MASSFLOW](#__RefHeading___Toc545769_3181922006) keyword defines the upstream river mass flow versus time tables for rivers, for when the River option has been activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
