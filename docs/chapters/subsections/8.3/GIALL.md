### GIALL -- Define Gi Values and PVT Properties versus Pressure

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The GIALL keyword defines the GI values and the associated RVGI, RSGI, BGGI and BOGI values as a function of pressure, for when the GI Pseudo Compositional option has been activated in the model via the GIMODEL keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
