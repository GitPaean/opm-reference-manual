#### Description

The [GIALL](#__RefHeading___Toc378111_1414963541) keyword defines the [GI](#__RefHeading___Toc372466_1414963541) values and the associated [RVGI](#__RefHeading___Toc335902_516898843), [RSGI](#__RefHeading___Toc316970_516898843), [BGGI](#__RefHeading___Toc223319_1539708736) and [BOGI](#__RefHeading___Toc223321_1539708736) values as a function of pressure, for when the [GI](#__RefHeading___Toc372466_1414963541) Pseudo Compositional option has been activated in the model via the [GIMODEL](#__RefHeading___Toc383678_1414963541) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
