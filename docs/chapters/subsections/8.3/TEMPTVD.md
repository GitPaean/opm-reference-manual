#### Description

[TEMPTVD](#__RefHeading___Toc1226279_4250154414) activates the Temperature Flux Limited Transport option in the commercial simulator, to reduce numerical dispersion for when either the [TEMP](#__RefHeading___Toc146397_3544483072) or [THERMAL](#__RefHeading___Toc137276_650382403) keywords in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section have been declared.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
