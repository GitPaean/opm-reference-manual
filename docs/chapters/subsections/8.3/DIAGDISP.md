#### Description

This keyword, [DIAGDISP](#__RefHeading___Toc308985_1539708736), activates the alternate form of tracer dispersion matrix for when the Tracer facility has been activated by the [TRACERS](#__RefHeading___Toc76509_327352552) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
