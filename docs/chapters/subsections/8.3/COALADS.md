#### Description

The [COALADS](#__RefHeading___Toc248981_3519154785) keyword defines the gas and solvent relative adsorption tables for when the coal phase has been activated via the [COAL](#__RefHeading___Toc234580_3519154785) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
