#### Description

This keyword, [PCW32D](#__RefHeading___Toc277129_2928331029), enables the water-oil capillary pressure data to be entered as a function of both oil and gas saturations. The keyword should be used in conjunction with the [SWF32D](#__RefHeading___Toc1186403_4250154414) keyword in the [PROPS](#__RefHeading___Toc39329_784232322) section. See also the [PCG32D](#__RefHeading___Toc271077_2928331029) keyword in the [PROPS](#__RefHeading___Toc39329_784232322) section that provides similar functionality for the gas-oil capillary pressure data.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
