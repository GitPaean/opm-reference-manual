#### Description

The [QHRATING](#__RefHeading___Toc624655_501926209) keyword defines a river's mass flow rate versus depth tables, for when the River option has been activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
