#### Description

This keyword, [EPSDBGS](#__RefHeading___Toc238206_803326780), defines the end-point debug data for multiple grid blocks that should be written to the [DEBUG](#__RefHeading___Toc52485_2479612490) file (\*.DBG) for when the End-Point Scaling option has been activated by the [ENDSCALE](#__RefHeading___Toc68146_2267116897) keyword in the RONSPEC section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
