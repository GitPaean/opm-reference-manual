### LKRW -- End-Point Scaling of Grid Cell Krw(Sw =1.0) (Low Salinity and Oil Wet)

#### Description

LKRW defines the scaling parameter at the maximum oil relative permeability value (SWU), that is for Sw = 1.0, for all the cells in the model via an array, for when the Low Salt option and the End-point Scaling options has been activated by the LOWSALT and the ENDSCALE keywords in the RUNSPEC section. The data is used to scale the water relative permeability in the low salinity oil wet water relative permeability saturation tables.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
