### MAXVALUE -- Sets a Maximum Value for an Array Element

The [MAXVALUE](#__RefHeading___Toc296601_1576177388) keyword sets a maximum value for the specified array or part of an array. The constant can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [MAXVALUE](#__RefHeading___Toc296601_1576177388) keyword is being used.
