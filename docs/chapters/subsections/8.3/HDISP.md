#### Description

The [HDISP](#__RefHeading___Toc301667_870710203) keyword is combined with three character tracer name, specified by the [TRACER](#__RefHeading___Toc121485_83452205) keyword in the [PROPS](#__RefHeading___Toc39329_784232322) section, to define the tracer's mechanical dispersivity parameters.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
