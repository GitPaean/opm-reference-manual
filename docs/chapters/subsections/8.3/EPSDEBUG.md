#### Description

This keyword, [EPSDEBUG](#__RefHeading___Toc238208_803326780), defines the end-point debug data for individual grid blocks that should be written to the Debug file (\*.DBG) for when the End-Point Scaling option has been activated by the [ENDSCALE](#__RefHeading___Toc68146_2267116897) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
