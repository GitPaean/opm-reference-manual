### EPSDEBUG -- Write End-Point Debug Data to the DEBUG File (Individual)

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword, EPSDEBUG, defines the end-point debug data for individual grid blocks that should be written to the Debug file (\*.DBG) for when the End-Point Scaling option has been activated by the ENDSCALE keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
