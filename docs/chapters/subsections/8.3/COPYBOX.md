### COPYBOX -- Copy Array Data Defined by a Box

The [COPYBOX](#__RefHeading___Toc297205_3519154785) keyword copies an array (or part of an array) to part of the same array. The array can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [COPYBOX](#__RefHeading___Toc297205_3519154785) keyword is being used.

See [[COPYBOX -- Copy Array Data Defined by a Box](#anchor-1)](#6.3.24.COPYBOX – Copy Array Data Defined by a Box|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
