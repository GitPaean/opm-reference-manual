#### Description

The [DPKRMOD](#__RefHeading___Toc204914_1772380413) keyword can be used to modify the matrix oil relative permeability data (oil-water, oil-gas) and the scaling of the fracture to matrix relative permeabilities, for dual porosity runs for when a Dual Porosity model has been activated by either the [DUALPORO](#__RefHeading___Toc241173_1772380413) or [DUALPERM](#__RefHeading___Toc241171_1772380413) keywords in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
