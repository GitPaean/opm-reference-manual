#### Description

This keyword activates the directional transmissibility multipliers for the [ROCKTAB](#__RefHeading___Toc107256_3812137098) keyword. This results in two additional columns being inputted on the [ROCKTAB](#__RefHeading___Toc107256_3812137098) keyword. This feature is currently not supported in OPM Flow.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
