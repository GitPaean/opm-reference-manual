#### Description

This keyword, [DISPERSE](#__RefHeading___Toc184339_1772380413), defines the dispersion tables for when the Dispersion option has been activated via declaring the dimensions of the [DISPERSE](#__RefHeading___Toc184339_1772380413) tables using the [DISPDIMS](#__RefHeading___Toc184279_1772380413) keyword and activating the Tracer option via the [TRACERS](#__RefHeading___Toc76509_327352552) keyword.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
