#### Description

[ALKADS](#__RefHeading___Toc192784_2366741785) defines the alkaline adsorption functions for when the alkaline model has been activated via the [ALKALINE](#__RefHeading___Toc197467_2366741785) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
