#### Description

The [HYDRHEAD](#__RefHeading___Toc195463_2135714711) keyword defines the hydraulic head reference data for when the hydraulic head information is requested to be written out via one on the [SUMMARY](#__RefHeading___Toc43949_784232322) keywords (BHD, BHDF, etc.) in the [SUMMARY](#__RefHeading___Toc43949_784232322) section, or to the [RESTART](#__RefHeading___Toc135629_1317547213) file via the HYDH or HYDHFW variables on the [RESTART](#__RefHeading___Toc135629_1317547213) keyword

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
