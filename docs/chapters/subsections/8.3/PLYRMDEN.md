#### Description

The [PLYRMDEN](#__RefHeading___Toc252003_501926209) keyword defines the in situ rock density at reservoir conditions by individual cell, for when the Polymer option has been activated by the [POLYMER](#__RefHeading___Toc38609_2267116897) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section. [PLYRMDEN](#__RefHeading___Toc252003_501926209) should consist of an array of real positive values. See the [DENSITY](#__RefHeading___Toc45799_719036256) parameter on the [PLYROCK](#__RefHeading___Toc110216_2939291539) keyword in the [PROPS](#__RefHeading___Toc39329_784232322) section for setting the property for the whole grid.
