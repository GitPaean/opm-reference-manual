#### Description

The [NARROW](#__RefHeading___Toc287842_1841740821) keyword activates the Run Summary Narrow Column Output option, for when printed [SUMMARY](#__RefHeading___Toc43949_784232322) data has been requested by the [RUNSUM](#__RefHeading___Toc210156_2884651453) keyword in the [SUMMARY](#__RefHeading___Toc43949_784232322) section. The option increases the number of columns "printed on the page".

There is no data required for this keyword and there is no terminating "/" for this keyword.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
