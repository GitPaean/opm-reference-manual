### NARROW -- Activate Run Summary Narrow Column Output Option

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The NARROW keyword activates the Run Summary Narrow Column Output option, for when printed SUMMARY data has been requested by the RUNSUM keyword in the SUMMARY section. The option increases the number of columns "printed on the page".

There is no data required for this keyword and there is no terminating "/" for this keyword.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
