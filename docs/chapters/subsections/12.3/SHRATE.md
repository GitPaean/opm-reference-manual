### SHRATE -- Activate and Define the Polymer Shearing Logarithmic Parameters

This keyword activates the logarithm-based polymer shear thinning/thickening option and defines the shear rate constant. This keyword can only be used in conjunction with the [PLYSHLOG](#__RefHeading___Toc110220_2939291539) in the [PROPS](#__RefHeading___Toc39329_784232322) section

See [[SHRATE - Activate and Define the Polymer Shearing Logarithmic Parameters](#anchor-1)](#7.2.68.SHRATE - Activate and Define the Polymer Shearing Logarithmic Parameters|outline) in the [PROPS](#__RefHeading___Toc39329_784232322) section for a full description.
