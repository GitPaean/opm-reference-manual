#### Description

This keyword, [RPTHMG](#__RefHeading___Toc298067_516898843), either enables or disables history match output reporting to the history match file (\*.HMD) for the named group, for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
