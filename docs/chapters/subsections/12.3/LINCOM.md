### LINCOM -- Define Linear Combination Coefficients

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The LINCOM keyword defines the oil, gas and water coefficients for the Linear Combination facility which allows for a linear combination of the aforementioned phase rates and volumes to be used as targets and constraints in controlling group and well production and injection data. See also the LCUNIT in the PROPS section that defines the units for linear combination equation.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
