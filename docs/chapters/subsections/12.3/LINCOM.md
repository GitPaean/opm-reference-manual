#### Description

The [LINCOM](#__RefHeading___Toc287176_2843394514) keyword defines the oil, gas and water coefficients for the Linear Combination facility which allows for a linear combination of the aforementioned phase rates and volumes to be used as targets and constraints in controlling group and well production and injection data. See also the [LCUNIT](#__RefHeading___Toc225544_2843394514) in the [PROPS](#__RefHeading___Toc39329_784232322) section that defines the units for linear combination equation.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
