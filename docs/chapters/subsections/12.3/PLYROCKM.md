#### Description

The [PLYROCKM](#__RefHeading___Toc252007_501926209) keyword modifies rock properties entered via the [PLYCAMAX](#__RefHeading___Toc239634_501926209), [PLYKRRF](#__RefHeading___Toc245845_501926209), [PLYRMDEN](#__RefHeading___Toc252003_501926209), and [PLYROCK](#__RefHeading___Toc110216_2939291539) keywords in the [PROPS](#__RefHeading___Toc39329_784232322) section, for when the Polymer option has been activated by the [POLYMER](#__RefHeading___Toc38609_2267116897) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
