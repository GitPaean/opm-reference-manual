#### Description

The [GRDREACH](#__RefHeading___Toc477197_1414963541) keyword defines the location of grid blocks connecting to a previously defined river, for when the River option has been activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
