### SAVE -- Activate Output of a SAVE File for Fast Restarts

This keyword activates output of a [SAVE](#__RefHeading___Toc55597_1778172979) file for fast restarts. There is no data required for this keyword.

See [[SAVE -- Activate Output of a SAVE File for Fast Restarts](#anchor-1)](#4.2.46.SAVE – Activates Output of a SAVE File for Fast Restarts|outline) in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section for a full description.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
