#### Description

The [APILIM](#__RefHeading___Toc402016_3174375363) keyword defines [API](#__RefHeading___Toc4422_421927891) Tracking grid block limits for when [API](#__RefHeading___Toc4422_421927891) Tracking has been activated via the [API](#__RefHeading___Toc4422_421927891) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section. The keyword enables the simulator to monitor the grid blocks outside the limits defined on the keyword, as well as to optionally contrain the values within a given range.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
