### APILIM -- Define API Tracking Grid Block Limits

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The APILIM keyword defines API Tracking grid block limits for when API Tracking has been activated via the API keyword in the RUNSPEC section. The keyword enables the simulator to monitor the grid blocks outside the limits defined on the keyword, as well as to optionally contrain the values within a given range.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
