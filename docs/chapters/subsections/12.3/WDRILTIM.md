#### Description

[WDRILTIM](#__RefHeading___Toc134882_2055188184) defines the automatic drilling parameters used to describe the numbers of days taken to drill a well, the drilling status of the well, and status of other wells when drilling an automatically drilled well.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
