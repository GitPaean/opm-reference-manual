#### Description

The [AQUCWFAC](#__RefHeading___Toc173881_1310555686) keyword modifies the datum depth and pressure for all aquifers specified by the [AQUCHWAT](#__RefHeading___Toc417060_3174375363) keyword in the [SOLUTION](#__RefHeading___Toc43947_784232322) or [SCHEDULE](#__RefHeading___Toc43945_784232322) sections.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
