#### Description

The [WELPRI](#__RefHeading___Toc998138_487874538) keyword is used to re-assign a priority number to a well for when the [PRIORITY](#__RefHeading___Toc46726_327352552) keyword has been used in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section. The [PRIORITY](#__RefHeading___Toc46726_327352552) keyword activates the Well Priority option and defines the coefficients in the well priority equation; [WELPRI](#__RefHeading___Toc998138_487874538) keyword can be used to over write these calculated priority numbers

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
