### WELPRI -- Assign Well Priority

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The WELPRI keyword is used to re-assign a priority number to a well for when the PRIORITY keyword has been used in the SCHEDULE section. The PRIORITY keyword activates the Well Priority option and defines the coefficients in the well priority equation; WELPRI keyword can be used to over write these calculated priority numbers

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
