#### Description

The COPMPRPL keyword re-scales the fluid saturations of a well's connection to an [LGR](#__RefHeading___Toc55049_4106839650) grid block, for when the Local Grid Refinement ("LGR") option has been activated by the [LGR](#__RefHeading___Toc55049_4106839650) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
