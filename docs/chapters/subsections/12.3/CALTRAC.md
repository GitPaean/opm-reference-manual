#### Description

The [CALTRAC](#__RefHeading___Toc229773_3519154785) keyword is used to assign a gas calorfic value to a tracer, for when the Tracer option has been invoked by the [TRACER](#__RefHeading___Toc121485_83452205) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
