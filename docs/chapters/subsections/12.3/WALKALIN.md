#### Description

WAKALIN keyword defines the water injection alkaline concentration for water injection wells for when the surfactant and/or polymer models have been activated by the [SURFACT](#__RefHeading___Toc863854_4250154414), [SURFACTW](#__RefHeading___Toc863864_4250154414), or the [POLYMER](#__RefHeading___Toc38609_2267116897) keywords in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section, combined with the [ALKALINE](#__RefHeading___Toc197467_2366741785) keyword which is also in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
