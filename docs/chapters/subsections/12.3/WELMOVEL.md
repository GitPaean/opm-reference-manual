#### Description

This keyword, [WELMOVEL](#__RefHeading___Toc984245_487874538), moves a previously defined gloabal well into a previously declared Local Grid Refinement ("LGR"), in a [RESTART](#__RefHeading___Toc135629_1317547213) run. The keyword should only be used in [RESTART](#__RefHeading___Toc135629_1317547213) runs.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
