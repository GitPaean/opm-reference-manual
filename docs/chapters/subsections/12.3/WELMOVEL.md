### WELMOVEL -- Move Global Well Into an LGR

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword, WELMOVEL, moves a previously defined gloabal well into a previously declared Local Grid Refinement ("LGR"), in a RESTART run. The keyword should only be used in RESTART runs.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
