### NETCOMPA -- Define Automatic Compressors (Extended Network)

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The NETCOMPA keyword defines automatic compressors in an extended network, for when the Extended Network option has been activated by the NETWORK keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
