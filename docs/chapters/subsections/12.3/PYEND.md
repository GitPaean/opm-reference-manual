### PYEND -- End the Definition of a PYINPUT Section

The [PYINPUT](#__RefHeading___Toc356358_1898124664) and [PYEND](#__RefHeading___Toc356356_1898124664) keywords are a part of OPM Flow's Python scripting facility that processes standard Python commands that can be used to manipulate and define the simulators input parameters during processing of the input deck. The main purpose of the facility is to script the construction of the various keywords. A [PYINPUT](#__RefHeading___Toc356358_1898124664) Definition Section is terminated by a [PYEND](#__RefHeading___Toc356356_1898124664) keyword on a separate single line.

See [[PYEND -- End the Definition of a PYINPUT Section](#anchor-1)](#6.3.166.PYEND – End the Definition of PYINPUT Commands|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
