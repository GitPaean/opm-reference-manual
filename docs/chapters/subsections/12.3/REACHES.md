### REACHES -- Define River Reaches Structure

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The REACHES keyword defines the reach structure of a previously characterized river system using the RIVERSYS keyword in the SCHEDULE section. The River option must be activated via the RIVRDIMS keyword in the RUNSPEC section in order to use these keywords.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
