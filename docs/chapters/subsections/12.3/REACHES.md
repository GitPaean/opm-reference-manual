#### Description

The [REACHES](#__RefHeading___Toc643464_501926209) keyword defines the reach structure of a previously characterized river system using the [RIVERSYS](#__RefHeading___Toc674955_501926209) keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section. The River option must be activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section in order to use these keywords.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
