### RIVERSYS -- Define River System (Branch Structure and Boundary Conditions)

[RIVERSYS](#__RefHeading___Toc674955_501926209) defines a river system by specifying the branch structure of the river together with the branch\'s associated boundary conditions, for when the River option has been activated via the [RIVRDIMS](#__RefHeading___Toc681201_501926209) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.
