### RIVERSYS -- Define River System (Branch Structure and Boundary Conditions)

RIVERSYS defines a river system by specifying the branch structure of the river together with the branch\'s associated boundary conditions, for when the River option has been activated via the RIVRDIMS keyword in the RUNSPEC section.

See [RIVERSYS - Define River System (Branch Structure and Boundary Conditions)](#anchor-1) in the SOLUTION section for a full description.
