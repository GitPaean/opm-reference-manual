#### Description

The [GASFDELC](#__RefHeading___Toc493662_1414963541) keyword defines how the field's gas deliverability calculation should be performed for when the Gas Field Operations option has been activated by the [GASFIELD](#__RefHeading___Toc195426_1190369742) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
