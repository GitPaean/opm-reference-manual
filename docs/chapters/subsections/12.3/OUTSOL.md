### OUTSOL -- Define Data to be Written to the RESTART File (Retired)

This keyword defines the data and frequency of the data to be written to the [RESTART](#__RefHeading___Toc135629_1317547213) file at each requested restart point. The keyword has been replaced by the [RPTRST](#__RefHeading___Toc210154_2884651453) keyword in the [SOLUTION](#__RefHeading___Toc43947_784232322) and [SCHEDULE](#__RefHeading___Toc43945_784232322) sections and is therefore considered retired.

See [[OUTSOL -- Define Data to be Written to the RESTART File (Retired)](#anchor-1)](#10.3.41.OUTSOL – Define Data to be Written to the RESTART File (Retired)|outline) in the [SOLUTION](#__RefHeading___Toc43947_784232322) section for a full description.
