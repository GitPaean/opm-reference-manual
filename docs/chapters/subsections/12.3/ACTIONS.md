### ACTIONS -- Define Action Conditions and Command Processing (Well Segments)

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The ACTIONS keyword defines a series of conditions that invoke run time processing of ACTION functions and is similar to executing a run time script for conditions and variables associated with well segments.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped; use the ACTIONX keyword instead.
