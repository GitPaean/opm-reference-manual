#### Description

The [DELAYACT](#__RefHeading___Toc303833_1539708736) keyword defines a series of keywords that should be executed after an [ACTION](#__RefHeading___Toc148342_63720426) keyword has been triggered by the [ACTION](#__RefHeading___Toc148342_63720426), [ACTIONG](#__RefHeading___Toc152219_2992482751), [ACTIONR](#__RefHeading___Toc152221_2992482751), [ACTIONW](#__RefHeading___Toc152225_2992482751), [ACTIONS](#__RefHeading___Toc152223_2992482751), or [ACTIONX](#__RefHeading___Toc152227_2992482751) keywords.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
