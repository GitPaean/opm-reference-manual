### PYINPUT -- Define the Start of a PYINPUT Section

The [PYINPUT](#__RefHeading___Toc356358_1898124664) and [PYEND](#__RefHeading___Toc356356_1898124664) keywords are a part of OPM Flow's Python scripting facility that processes standard Python commands that can be used to manipulate and define the simulators input parameters during processing of the input deck. The main purpose of the facility is to script the construction of the various keywords used by the simulator. [PYINPUT](#__RefHeading___Toc356358_1898124664) declares the start of a [PYINPUT](#__RefHeading___Toc356358_1898124664) Definition Section on a single separate line, which is then followed by various standard Python commands, with one command per line.

See [[PYINPUT -- Define the Start of a PYINPUT Section](#anchor-1)](#6.3.167.PYINPUT – Define Python Based Action Conditions and Command Processing|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
