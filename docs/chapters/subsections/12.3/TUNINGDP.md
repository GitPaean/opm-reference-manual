#### Description

The [TUNINGDP](#__RefHeading___Toc50286_327352552) keyword defines the parameters used for controlling the commercial simulator's numerical convergence parameters. This keyword is similar to the [TUNING](#__RefHeading___Toc146744_4203985108) keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section, but the defaults on this keyword are optimized for high throughput runs.

This keyword is generally ignored by OPM Flow; however, the simulator can be instructed to read some parameters from the [TUNINGDP](#__RefHeading___Toc50286_327352552) keyword if the command line option \--enable-tuning=true has been used (see section [[2.2](#anchor-2)](#2.2.Running OPM Flow 2018-10 |outline)[](#2.2.Running OPM Flow 2018-10 |outline)[[Running OPM Flow 2023-04 From The Command Line](#anchor-2)](#2.2.Running OPM Flow 2018-10 |outline)).
