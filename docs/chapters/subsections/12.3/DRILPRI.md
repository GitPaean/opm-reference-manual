#### Description

This keyword, [DRILPRI](#__RefHeading___Toc215311_1772380413), defines the prioritized drilling queue priority parameters used in the priority formulae for this type of drilling queue.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
