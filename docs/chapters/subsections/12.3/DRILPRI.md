### DRILPRI -- Define Prioritized Drilling Queue Priority Parameters

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword, DRILPRI, defines the prioritized drilling queue priority parameters used in the priority formulae for this type of drilling queue.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
