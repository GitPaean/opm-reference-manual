#### Description

This keyword, [WCUTBACT](#__RefHeading___Toc437067_2026549522), defines a production well's cutback limits and parameters based on the named produced tracer from the well. See also the [GCUTBACT](#__RefHeading___Toc437067_20265495221) keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section that provides similar functionality for groups.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
