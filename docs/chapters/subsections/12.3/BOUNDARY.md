### BOUNDARY -- Define a Boundary Box for Printing

The [BOUNDARY](#__RefHeading___Toc161697_1314821763) keyword defines a rectangular grid for printing various arrays to the output print file (\*.PRT); thus, avoiding printing all the elements in the selected array.

See [[BOUNDARY -- Define a Boundary Box for Printing](#anchor-1)](#6.3.12.BOUNDARY – Define a Boundary Box for Printing|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
