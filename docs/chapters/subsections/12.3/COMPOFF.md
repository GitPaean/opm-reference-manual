#### Description

The [COMPOFF](#__RefHeading___Toc287522_3519154785) keyword deactivates network automatic compressors defined via the [GASFCOMP](#__RefHeading___Toc195056_2110098628) keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
