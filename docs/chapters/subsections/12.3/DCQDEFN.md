#### Description

The [DCQDEFN](#__RefHeading___Toc298755_1539708736) keyword defines the DCQ units to be rate or energy (calorific value) when using the Gas Field Operation model and the Gas Calorific Value control option. The gas DCQ rates are controlled by the [GASYEAR](#__RefHeading___Toc493664_1414963541), [GASPERIO](#__RefHeading___Toc206080_1190369742), [GDCQ](#__RefHeading___Toc229883_156692946), [GASFTARG](#__RefHeading___Toc200746_1190369742) or [GASFDECR](#__RefHeading___Toc184697_1190369742) keywords in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
