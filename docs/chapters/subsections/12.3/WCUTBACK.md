### WCUTBACK -- Define Well Cutback Limits and Parameters

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword, WCUTBACK, defines a well's cutback limits and parameters for both production and injection wells. See also the GCUTBACK keyword in the SCHEDULE section that provides similar functionality for groups.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
