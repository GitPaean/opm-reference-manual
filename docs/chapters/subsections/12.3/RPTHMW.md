### RPTHMW -- Define Well History Match Gradient Reporting (Wells)

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword, RPTHMG, either enables or disables history match output reporting to the history match file (\*.HMD) for the named well, for when the History Match Gradient option has been activated by the HMDIMS keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
