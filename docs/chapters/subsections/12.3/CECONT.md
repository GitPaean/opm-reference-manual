#### Description

This keyword, [CECONT](#__RefHeading___Toc239382_3519154785), sets the tracer economic cut-off criteria for a well's connection to the simulation grid.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
