#### Description

The [GRADGRUP](#__RefHeading___Toc427249_1414963541) keyword defines the [SUMMARY](#__RefHeading___Toc43949_784232322) field and group vectors that should be written to the History Match Gradient output file, for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
