### PLYVISCS -- Define Polymer-Salt Viscosity Scaling Factors

PLYSVISCS defines the polymer-salt viscosity scaling factor tables applied to pure water that are used to determine the viscosity of a polymer-salt mixture with respect to increasing polymer saturation within a grid block. The polymer option must be activated by the POLYMER keyword, as well as the brine phase declared by the BRINE keyword in the RUNSPEC section in order to use this keyword. However the ECLM keyword in the RUNSPEC must not be used with this keyword.

See [PLYVISCS -- Define Polymer-Salt Viscosity Scaling Factors](#anchor-1) in the PROPS section for a full description.
