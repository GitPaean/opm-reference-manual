#### Description

This keyword, [WCONINJP](#__RefHeading___Toc537532_38555213), defines well injection targets and constraints for pattern flood wells. The keyword is similar to the [WCONINJE](#__RefHeading___Toc146750_4203985108) keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section except that the injection control is applied to a group of wells defined by the first record of this keyword, combined with a second record that defines the wells in the pattern and their contribution to the pattern.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
