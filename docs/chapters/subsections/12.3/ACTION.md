#### Description

The [ACTION](#__RefHeading___Toc148342_63720426) keyword defines a series of conditions that invoke run time processing of [ACTION](#__RefHeading___Toc148342_63720426) functions and is similar to executing a run time script for conditions and variables at the field level.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped; use the [ACTIONX](#__RefHeading___Toc152227_2992482751) keyword instead.
