#### Description

This keyword, [DUMPCUPL](#__RefHeading___Toc246382_1772380413), activates output to the Reservoir Coupling file from the reservoir coupling file in the master run for when reservoir coupling is invoked by the [GRUPMAST](#__RefHeading___Toc488185_1414963541) and [SLAVES](#__RefHeading___Toc1268538_516898843) keywords in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
