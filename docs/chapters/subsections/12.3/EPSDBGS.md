### EPSDBGS -- Write End-Point Debug Data to the DEBUG File (Multiple)

This keyword, EPSDBGS, defines the end-point debug data for multiple grid blocks that should be written to the debug file (\*.DBG) for when the End-Point Scaling option has been activated by the ENDSCALE keyword in the RONSPEC section.

See [EPSDBGS - Write End-Point Debug Data to the DEBUG File (Multiple)](#anchor-1) in the PROPS section for afull description.
