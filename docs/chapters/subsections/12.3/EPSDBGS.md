### EPSDBGS -- Write End-Point Debug Data to the DEBUG File (Multiple)

This keyword, [EPSDBGS](#__RefHeading___Toc238206_803326780), defines the end-point debug data for multiple grid blocks that should be written to the debug file (\*.DBG) for when the End-Point Scaling option has been activated by the [ENDSCALE](#__RefHeading___Toc68146_2267116897) keyword in the RONSPEC section.

See [[EPSDBGS - Write End-Point Debug Data to the DEBUG File (Multiple)](#anchor-1)](#8.3.36.EPSDBGS - Define the End-Point Debug Data to the DEBUG File (Multiple)|outline) in the [PROPS](#__RefHeading___Toc39329_784232322) section for afull description.
