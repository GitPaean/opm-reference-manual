#### Description

The [DYNAMICR](#__RefHeading___Toc251605_1772380413) keyword marks the start of a Dynamic Region section and defines the parameters used for Dynamic Regions that allows for property and reporting regions to vary as the run progresses, based on the parameters and logic defined by this keyword and section. A Dynamic Region section is terminated by the [ENDDYN](#__RefHeading___Toc230794_252421755) keyword in the [SOLUTION](#__RefHeading___Toc43947_784232322) or [SCHEDULE](#__RefHeading___Toc43945_784232322) sections.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
