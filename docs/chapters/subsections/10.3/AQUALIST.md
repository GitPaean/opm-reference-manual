#### Description

The [AQUALIST](#__RefHeading___Toc412035_3174375363) keyword defines named lists of analytic aquifers identified by aquifer numbers for greater readability in the output.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
