#### Description

The [AQANTRC](#__RefHeading___Toc169183_1310555686) keyword defines the initial tracer concentrations for analytic aquifers that have previously been defined by the AQCT keyword in the [GRID](#__RefHeading___Toc38674_784232322), [PROPS](#__RefHeading___Toc39329_784232322), or [SOLUTION](#__RefHeading___Toc43947_784232322) sections for Carter-Tracy analytical aquifers, or the AQFET and AQFETP keywords in the [SOLUTION](#__RefHeading___Toc43947_784232322) section for Fetkovich analytical aquifers.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
