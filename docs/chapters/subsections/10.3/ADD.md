### ADD -- Add a Constant to a Specified Array

The [ADD](#__RefHeading___Toc4412_421927891) keyword adds a constant to a specified array or part of an array. The constant can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [ADD](#__RefHeading___Toc4412_421927891) keyword is being applied.

See [[ADD -- Add a Constant to a Specified Array](#anchor-1)](#1.3.2.ADD – Add a Constant to a Specified Array|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
