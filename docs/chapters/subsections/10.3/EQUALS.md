### EQUALS -- Sets a Specified Array to a Constant

The [EQUALS](#__RefHeading___Toc296597_1576177388) keyword sets a specified array or part of an array to a constant. The constant can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [EQUALS](#__RefHeading___Toc296597_1576177388) keyword is being used.

See [[EQUALS -- Sets a Specified Array to a Constant](#anchor-1)](#1.3.10.EQUALS – Sets a Constant to a Specified Array|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
