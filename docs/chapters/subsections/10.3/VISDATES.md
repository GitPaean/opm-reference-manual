#### Description

The [VISDATES](#__RefHeading___Toc654279_487874538) keyword defines External Reservoir Geo-Mechanics [VISAGE](#__RefHeading___Toc654274_487874538) option stress dates. The keyword should not be used in input decks as the associated data is generated by an external program.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
