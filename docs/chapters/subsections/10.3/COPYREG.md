### COPYREG -- Copy an Array to Another Array based on a Region Number

The [COPYREG](#__RefHeading___Toc296589_1576177388) keyword copies a specified array or part of an array based on cells with a specific region number to another array. The region number array can be [FLUXNUM](#__RefHeading___Toc45781_719036256), [MULTNUM](#__RefHeading___Toc61329_2752266063) or [OPERNUM](#__RefHeading___Toc67857_718313858) and these arrays must be defined and be available before the [COPYREG](#__RefHeading___Toc296589_1576177388) keyword is read by the simulator. The property arrays can be real or integer depending on the property array type; however, the property arrays that can be operated on is dependent on which section the [COPYREG](#__RefHeading___Toc296589_1576177388) keyword is being used.

See [[COPYREG -- Copy an Array to Another Array based on a Region Number](#anchor-1)](#4.3.12.COPYREG|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
