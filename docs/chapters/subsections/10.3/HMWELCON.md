#### Description

This keyword, [HMWELCON](#__RefHeading___Toc256883_373485663), defines the history match gradient parameters for well connection factors and connection skins, for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section. Wells must be specified using the WELPSECS keyword in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section and their connections defined by the [COMPDAT](#__RefHeading___Toc97651_3261743917) and/or [COMPDATL](#__RefHeading___Toc153110_63720426) keywords, also in the [SCHEDULE](#__RefHeading___Toc43945_784232322) section

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
