### AQUANCON -- Define Analytical Connections to the Grid

The [AQUANCON](#__RefHeading___Toc177536_3429068809) keyword defines how analytical aquifers are connected to the simulation grid, this includes Carter-Tracy and Fetkovich analytical aquifers, both of which are implemented in OPM Flow.

See [[AQUANCON -- Define Analytical Connections to the Grid](#anchor-1)](#6.3.5.AQUANCON – Define Analytical Connections to the Grid|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
