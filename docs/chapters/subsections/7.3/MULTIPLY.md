### MULTIPLY -- Multiply a Specified Array by a Constant

The [MULTIPLY](#__RefHeading___Toc296609_1576177388) keyword multiplies a specified array or part of an array by a constant. The constant can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the keyword is being used.

See [[MULTIPLY -- Multiply a Specified Array by a Constant](#anchor-1)](#1.3.12.MULTIPLY – Multiply a Specified Array by a Constant|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
