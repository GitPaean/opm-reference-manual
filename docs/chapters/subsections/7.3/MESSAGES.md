### MESSAGES -- Define Message Print Limits and Stop Limits

The [MESSAGES](#__RefHeading___Toc61634_2479612490) keyword defines the print and stops levels for various messages. The "print limits" set the maximum number of messages that will be printed, after which no more messages will be printed and the "stop limits" terminate the run when these limits are exceeded. There are six levels of message that increase in severity from informative all the way to programming errors, as outlined in [Table 4.5](#anchor-1).

See [[MESSAGES -- Define Message Print Limits and Stop Limits](#anchor-2)](#3.1.7.MESSAGES – Defines Message Print Limits and Stop Limits|outline) in the GLOBAL section for a full description.
