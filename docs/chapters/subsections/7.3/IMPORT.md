### IMPORT -- Import Grid File Data at the Current Position

The [IMPORT](#__RefHeading___Toc539691_2135714711) keyword informs the simulator to continue reading input data from the specified [IMPORT](#__RefHeading___Toc539691_2135714711) file. When the end of the [IMPORT](#__RefHeading___Toc539691_2135714711) file is reached, input data is read from the next keyword in the current file. Normally [IMPORT](#__RefHeading___Toc539691_2135714711) files are generated by grid pre-processing software and the keyword allows for both formatted and unformatted (binary) files to be loaded.

See [[IMPORT -- Import Grid File Data at the Current Position](#anchor-1)](#6.3.84.IMPORT – Import Grid File at the Current Position|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
