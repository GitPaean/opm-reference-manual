### EQUALREG -- Sets an Array to a Constant by Region Number

The [EQUALREG](#__RefHeading___Toc296593_1576177388) keyword sets a specified array to a constant for cells with a specific region number. The region number array can be [FLUXNUM](#__RefHeading___Toc45781_719036256), [MULTNUM](#__RefHeading___Toc61329_2752266063) or [OPERNUM](#__RefHeading___Toc67857_718313858) and these arrays must be defined and be available before the [EQUALREG](#__RefHeading___Toc296593_1576177388) keyword is read by the simulator. The constant can be real or integer depending on the property array type; however, the property arrays that can be operated on is dependent on which section the [EQUALREG](#__RefHeading___Toc296593_1576177388) keyword is being used.

See [[EQUALREG -- Sets an Array to a Constant by Region Number](#anchor-1)](#4.3.23.EQUALREG – Sets a Constant for an Array based on a Region Number|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
