### MULTTHT- -- Multiply Cell Transmissibility in the -Theta Direction

MULTTHT- multiples the transmissibility between two cell faces in the -Theta direction for all the cells in the model via an array, that is the keyword sets the transmissibility multiplier of block (I, J, K) between the cells (I, J-1, K) and (I, J, K). The keyword should only be used with radial grids, as declared by the RADIAL keyword in the RUNSPEC section.

See [MULTTHT- - Multiply Cell Transmissibility in the -Theta Direction](#anchor-1) in the GRID section for a full description.
