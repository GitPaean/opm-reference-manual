### DIFFTHT -- Define Grid Block Theta Direction Diffusivity Values

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The DIFFTHT keyword defines the theta direction diffusivity values for cells in the current input box for when the Diffusivity option has been activated by the DIFFUSE keyword in the RUNSPEC section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
