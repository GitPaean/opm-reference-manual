### MINVALUE -- Set a Minimum Value for an Array Element

The [MINVALUE](#__RefHeading___Toc296605_1576177388) keyword sets a minimum value for the specified array or part of an array. The constant can be real or integer depending on the array type; however, the arrays that can be operated on is dependent on which section the [MINVALUE](#__RefHeading___Toc296605_1576177388) keyword is being used.

See [[MINVALUE -- Set a Minimum Value for an Array Element](#anchor-1)](#5.3.36.MINPV – Define Minimum Grid Block Pore Volume Threshold|outline) in the [GRID](#__RefHeading___Toc38674_784232322) section for a full description.
