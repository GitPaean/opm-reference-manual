#### Description

The [MESSAGE](#__RefHeading___Toc551671_3181922006) keyword outputs a user message to the terminal, as well as to the print (\*.PRT) and debug (\*.DBG) files. Note this is different to the [MESSAGES](#__RefHeading___Toc61634_2479612490) keyword, that defines OPM Flows message print limits and stop limits generated by the simulator.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
