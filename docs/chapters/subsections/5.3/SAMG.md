### SAMG -- Activate Algebraic Multi-Grid Linear Solver

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

This keyword actives the algebraic multi-grid linear solve; note this solver is not available to the general public in the commercial simulator.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.

See section [Running OPM Flow From The Command Line](#running-opm-flow-from-the-command-line) on how to invoke various numerical schemes via the OPM Flow command line interface.
