#### Description

[RIVRDIMS](#__RefHeading___Toc681201_501926209) defines the river system array dimensions used with the [REACHES](#__RefHeading___Toc643464_501926209) keyword and other river keywords in the [SOLUTION](#__RefHeading___Toc43947_784232322) and [SCHEDULE](#__RefHeading___Toc43945_784232322) sections. The keyword also enables the River option.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
