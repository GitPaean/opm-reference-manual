#### Description

The BPDIMS keyword defines the dimensions of the interpolated grid block quantities for the BPR_X, BHD_X,BHDF_X, BSCN_X and BCTRA_X, etc. variables declared in the [SUMMARY](#__RefHeading___Toc43949_784232322) section.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
