#### Description

The [VISAGE](#__RefHeading___Toc654274_487874538) keyword activates the External Reservoir Geo-Mechanics VISAGE option. The keyword should not be used in input decks as the associated data is generated by an external program.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
