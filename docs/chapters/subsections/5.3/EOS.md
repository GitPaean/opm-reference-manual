#### Description

The [EOS](#REF_HEADING_KEYWORD_EOS_5_3) keyword specifies the Equation Of State (EOS) to be used for each EOS region. The keyword should only be used if the compositional mode has been requested using the [COMPS](#__RefHeading___Toc27871_3671211675 Copy 1) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

See [[EOS -- Specify Equation of State](#anchor-2)](#0.0.0.EOS) in the [PROPS](#__RefHeading___Toc39329_784232322) section for a full description.
