#### Description

The [PARTTRAC](#__RefHeading___Toc264979_2928331029) keyword activates the Partitioned Tracer option and defines the maximum number of partitioned tracers, the number of [TRACERKP](#__RefHeading___Toc1279807_4250154414) or [TRACERKM](#__RefHeading___Toc1279805_4250154414) partitioning tables in the [PROPS](#__RefHeading___Toc39329_784232322) section, and the maximum number of number of rows in the [TRACERKP](#__RefHeading___Toc1279807_4250154414) or [TRACERKM](#__RefHeading___Toc1279805_4250154414) partitioning tables.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
