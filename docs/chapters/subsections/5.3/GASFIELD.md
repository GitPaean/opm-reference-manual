#### Description

The [GASFIELD](#__RefHeading___Toc195426_1190369742) keyword activates and specifies the Gas Field Operations options and determines if extended compressors are present in the run and if the expedited first pass DCQ calculation should be used.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
