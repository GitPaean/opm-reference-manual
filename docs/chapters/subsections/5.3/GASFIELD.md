### GASFIELD -- Define Gas Field Operations Options

| RUNSPEC | GRID | EDIT | PROPS | REGIONS | SOLUTION | SUMMARY | SCHEDULE |
| --- | --- | --- | --- | --- | --- | --- | --- |

#### Description

The GASFIELD keyword activates and specifies the Gas Field Operations options and determines if extended compressors are present in the run and if the expedited first pass DCQ calculation should be used.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
