#### Description

This keyword, [RPTHMD](#__RefHeading___Toc291769_516898843), defines the options and level of history match output that should be written to history match file (\*.HMD), for when the History Match Gradient option has been activated by the [HMDIMS](#__RefHeading___Toc207753_4219267791) keyword in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section.

This keyword is not supported by OPM Flow but has no effect on the results so it will be ignored.
