#### Description

The [MESSSRVC](#__RefHeading___Toc563352_3181922006) keyword activates or deactivates output to the database message file (\*.DBPRTX). The file contains all the messages from run in binary format and is used in some post-processing software to annotate production line plots from the run.

This keyword is not supported by OPM Flow but would change the results if supported so the simulation will be stopped.
