# <a id="-refheading-toc1616-3043861001"></a>PROPS SECTION

## <a id="-refheading-toc37624-3701168388"></a>Introduction

The [PROPS](#-refheading-toc39329-784232322) section is the section that contains the fluid property keywords used to define the PVT behavior of the fluids in the model and therefore the data is fluid type dependent.  For example, if the oil phase has been activated in the [RUNSPEC](#-refheading-toc55591-1778172979) section via the [OIL](#-refheading-toc97439-1778172979) keyword then one of the oil PVT keywords needs to be defined in this section. The other main type of data required for the [PROPS](#-refheading-toc39329-784232322) section is the saturation tables that govern how the various fluids flow in the model relative to the other phases. Again, the saturation tables are fluid type dependent. 
Various other fluid and rock property data are also entered in this section including rock compressibility via the [ROCK](#-refheading-toc45809-719036256) keyword for the standard rock compressibility model, and rather unusually, the Carter\-Tracy influence function entered via the [AQUTAB](#-refheading-toc110746-846947960) keyword.
