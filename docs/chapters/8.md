# PROPS SECTION

## Introduction

The [PROPS](#__RefHeading___Toc39329_784232322) section is the section that contains the fluid property keywords used to define the PVT behavior of the fluids in the model and therefore the data is fluid type dependent. For example, if the oil phase has been activated in the [RUNSPEC](#__RefHeading___Toc55591_1778172979) section via the [OIL](#__RefHeading___Toc97439_1778172979) keyword then one of the oil PVT keywords needs to be defined in this section. The other main type of data required for the [PROPS](#__RefHeading___Toc39329_784232322) section is the saturation tables that govern how the various fluids flow in the model relative to the other phases. Again, the saturation tables are fluid type dependent.

Various other fluid and rock property data are also entered in this section including rock compressibility via the [ROCK](#__RefHeading___Toc45809_719036256) keyword for the standard rock compressibility model, and rather unusually, the Carter-Tracy influence function entered via the [AQUTAB](#__RefHeading___Toc110746_846947960) keyword.
