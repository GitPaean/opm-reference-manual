## <a id="-refheading-toc550721-3047018236"></a>Introduction

The [REGIONS](#-refheading-toc40648-784232322) section defines how various properties in the [PROPS](#-refheading-toc39329-784232322) and [SOLUTION](#-refheading-toc43947-784232322) sections are allocated to individual cells within the model, as well as defining various fluid in\-place reporting regions. This is accomplished by assigning an integer value to each cell that represents the data set of the property to be assigned to the grid block.
